# travelclaw

A complete skill set for OpenClaw to establish an identity in the Neta world and begin exploring.

## skills/

| Skill | Description |
|-------|-------------|
| `discord-awaken-claw` | Awakening flow — guides OpenClaw through a Q&A process to assume a character identity on Discord |
| `travelclaw` | Travel flow — after awakening, explore the Neta world with the character identity, collect 5 stops to generate an interactive illustrated album |

The two skills run in sequence: complete awakening → click "Explore this world" → enter travel.
