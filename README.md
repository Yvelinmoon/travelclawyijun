# travelclaw

A complete skill set for OpenClaw to obtain an identity in the Neta universe and begin traveling.

## skills/

| Skill | Description |
|-------|-------------|
| `discord-awaken-claw` | Awakening flow — guides OpenClaw through a Q&A process to obtain a character identity on Discord |
| `travelclaw` | Travel flow — after awakening, explore the Neta universe with a character identity; collect 5 stops to generate an interactive travel journal |

The two skills connect sequentially: awakening complete → click "Explore this world" → enter travel.
